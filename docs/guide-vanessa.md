# Guide Utilisateur — Portail Rescape 🌱

Bienvenue Vanessa ! Ce guide vous aide à prendre en main la gestion de votre association via le nouveau portail admin.

## 1. Connexion au Dashboard
- Allez sur [rescape.fr/admin/login](https://rescape.fr/admin/login) (URL à confirmer selon votre domaine).
- Utilisez vos identifiants fournis.
- **Mot de passe oublié ?** Un lien "Mot de passe oublié" vous permet désormais d'initier une réinitialisation.

## 2. Gestion des Actualités (Articles)
- **Liste :** Visualisez tous vos articles.
- **Statut :** Utilisez le switch pour publier ou passer en brouillon.
- **Nouveau :** Un titre accrocheur, une image de couverture, et votre contenu via l'éditeur riche.
- **Note :** Le champ "Image" a été unifié pour plus de simplicité.

## 3. Gestion des Événements
- **Statuts :** Vous pouvez désormais marquer un événement comme "Brouillon", "Publié" ou "Annulé".
- **Affichage :** Seuls les événements "Publiés" apparaissent sur le site public.

## 4. Newsletter
- Consultez la liste des inscrits dans l'onglet "Newsletter".
- Vous pouvez désabonner manuellement un membre si nécessaire (conformité RGPD).
- Un export CSV est disponible pour vos envois groupés.

## 5. Médiathèque
- Retrouvez toutes les images utilisées sur le site dans l'onglet "Médiathèque".
- Cela vous permet de voir d'un coup d'œil si des images manquent ou doivent être mises à jour.

## 6. Accessibilité & RGPD
- Le site inclut désormais une bannière de consentement cookies.
- Un lien "Aller au contenu principal" est disponible au clavier pour les personnes en situation de handicap.
